# Privacy and data protection statement

The original clinical dataset used in the AMIS-DynaFlow study cannot be publicly released due to institutional and ethical restrictions related to patient privacy.

This public package contains no directly identifiable patient information and no real hospital operational records. It provides:

- anonymized data schemas;
- public sample data for methodological demonstration;
- preprocessing scripts;
- model configuration files; and
- executable workflow scripts.

The sample dataset is not derived from individual patient records and must not be interpreted as real clinical data. It is provided solely to reproduce the computational workflow and validate the implementation pathway.

Prohibited content in this repository includes patient names, hospital numbers, national identification numbers, phone numbers, addresses, exact dates of birth, raw visit identifiers, database credentials, EHR/HIS credentials, and any unapproved clinical record export.
